# Godot TDS

Godot plugin for supporting TapTap Developer Services

在 Godot 中使用 TapTap 开发者服务

# Version

- OpenJDK 17.0.11
- TapSDK Android 3.29.2
- TapADN Android SDK 3.16.3.31
- Godot 4.2.2

# Feature

- [x] 登录
- [x] 合规认证
- [x] 内嵌动态
- [x] 悬浮窗
- [ ] 成就系统
- [ ] 礼包系统
- [ ] 排行榜
- [ ] 云存档
- [ ] 激励广告
